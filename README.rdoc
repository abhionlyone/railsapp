== README

The intention of this repo is to integrate/config generic services/gems such as devise,OmniAuth,AWS S3 etc.,

Specifications:

* Ruby version 2.3.1

* Rails version 4.2.7.1

* Database MySql

* Integrated Devise Authentication with confirmable email and Facebook login

* Configured PaperClip gem to upload files to S3

* Integrated figaro gem to save sensitive secret ids,Passwords for services like Database, Facebook, AWS S3 etc.,
